
# Zadonetske App v1.4.7

- Мобильное меню рендерится в **портале** в `<body>` — теперь это полноценный непрозрачный фуллскрин-лист (`100vw/100vh`), без просвечивания и с блокировкой скролла.
- Переключатель языков — только в хедере, без дублей в футере.
- Плавные темы (ethno/river/forest/minimal) через CSS-переменные.


## Запуск
```bash
npm i
npm run dev
```
